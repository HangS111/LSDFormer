# LSDFormer

## Install Dependencies
install the ultralytics package, including all requirements, in a Python>=3.8 environment with PyTorch>=1.8.  
    `pip install ultralytics`        
 

## Usages
model profiles path:  
        ultralytics-main\profiles
train and test files name:  
        train.py   
        test.py


## notices:
Before starting the model training, please first prepare the dataset and configure the dataset configuration file.
